# 查老师 - chalaoshi 离线版（Update to 2025-05-02）

## 文件结构

```
web/
├── teachers/     # 教师个人页面目录
|   └── ...       # 教师页面
├── index.html    # 主页面
└── README.md
```

## 使用方法

### 打开主页面
- 直接双击打开 `index.html` 文件
- 或在浏览器中打开该文件
- 支持完全离线使用

### 功能特性
- **教师搜索**: 快速搜索教师姓名
- **评分查看**: 查看教师评分和评论
- **课程GPA**: 查看课程GPA数据
